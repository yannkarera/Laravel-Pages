# Gestionnaire de Fruits - Projet Laravel

## Description

Ce projet est une application web développée avec **Laravel 8/9/10** (PHP 8.0+) qui permet de gérer une liste de fruits.  
Fonctionnalités principales :  

- Afficher tous les fruits enregistrés  
- Ajouter un nouveau fruit  
- Modifier ou supprimer un fruit existant  
- Afficher les détails d’un fruit  

L’application utilise **Blade** pour le rendu des pages, **Eloquent ORM** pour la base de données, et le système de routes Laravel pour naviguer entre les pages.

---

## Prérequis

Avant de lancer le projet, assurez-vous d’avoir installé :  

- **PHP >= 8.0**  
- **Composer** (gestionnaire de dépendances PHP)  
- **Serveur web** : Apache, Nginx ou serveur intégré Laravel (`php artisan serve`)  
- **Base de données** : MySQL (ou MariaDB, SQLite, PostgreSQL)
